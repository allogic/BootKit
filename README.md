# EfiGate

```
Usage:

 EfiGate [command]

Commands:
 setup          Setup EDK2 repository
 mkdrive        Make a hard drive image
 mount          Mount the hard drive image
 umount         Unmount the hard drive image
 update         Update BIOS and BOOTX64.EFI file
 boot           Boot QEMU
 help           Display this help message
```
